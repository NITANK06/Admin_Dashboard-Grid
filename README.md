1. Admin Dashboard Grid

A modern and responsive **Admin Dashboard Grid** built using React (Vite).  
This project demonstrates clean architecture, reusable components, CSS Grid layout, and form validation.



2. Features

- Responsive CSS Grid Layout
- Sidebar Navigation
- Header with Search Bar
- Dashboard Statistics Cards
- Users Data Table
- Add User Form with Validation
- Modular & Reusable Components
- Clean Code Structure



3. Tech Stack

- Frontend: React (Vite)
- Styling: CSS Grid & Flexbox
- Icons: React Icons
- Language: JavaScript (ES6+)

---





 4. Sidebar
Navigation links for Dashboard, Users, and Reports.

Contains search bar and admin profile section.

Reusable component to display dashboard statistics.

Displays user data in tabular format.

Form to add new users with:
- Required field validation
- Email format validation
- Error messages
- Form reset after successful submission
